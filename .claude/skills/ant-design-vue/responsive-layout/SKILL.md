
# Responsive & Adaptive Layout（响应式自适应布局）

你专注于 **Vue 3 + Ant Design Vue 4.x** 后台页面的响应式与自适应布局。

---

## When to Apply

- 新建页面布局结构时
- 兼容不同屏幕宽度时
- 优化现有页面的自适应表现时

---

## 兼容目标

以桌面端后台为主，兼容：1440、1366、1280、1024

窄屏下允许折行/换列/收起，但不允许主要操作消失。尽量避免整体横向滚动，表格列过多时只允许表格区域内部滚动。

---

## 布局规则

| 区域 | 规则 |
|------|------|
| 页面 | `a-row` + `a-col` 24 栅格，gutter 优先 16 或 24 |
| 查询区 | `flex` + `flex-wrap`，不写死成一行，窄屏自动折列 |
| 表格区 | 外层 `overflow-x: auto`，操作列保持可见 |
| 弹窗 | Modal 宽度区间控制，大表单用 Drawer，内容超限内部滚动 |
| 按钮区 | 窄屏自动换行，过多时折叠到更多菜单 |

## 样式建议

- 优先 `flex` / `grid`，使用 `clamp()` 和百分比控制尺寸
- 断点围绕：`1200px`、`992px`、`768px`

---

## 技术约束

- Vue 3 + `script setup`，JavaScript，Ant Design Vue 4.x
