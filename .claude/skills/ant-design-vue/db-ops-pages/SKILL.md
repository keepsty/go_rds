
# Database & Ops Admin Pages（数据库运维页面专项）

你专注于**数据库、工单、审批、运维类**后台页面开发。

---

## When to Apply

- 数据库工单页、审批流页
- SQL 审核 / 结果展示页
- 资源申请 / 扩容页
- 慢查询分析 / 监控告警页
- 环境配置 / 权限管理页

---

## 核心原则

1. 信息优先级明确，结果展示比装饰重要
2. 查询效率比视觉炫技重要
3. 操作风险要清晰提示
4. 结果页、详情页、审核页突出关键信息

---

## 页面专项规范

### 工单类
必须突出：工单状态、执行环境、风险等级、审批链路

### SQL 结果展示
重视可读性，SQL 文本需语法高亮、展示合适

### 审批流
必须突出：节点关系、当前节点、审批结论、各节点审批人/时间

### 风险与安全
- 环境明确区分：测试/预发/生产
- 危险操作 `danger` + `modal.confirm()` 双重确认
- 高风险按钮颜色、文案都要更直接

---

## 典型场景

数据库工单 / 审批流配置 / SQL 审核 / 资源申请 / 扩容 / 慢查询 / 监控告警 / 变更记录 / 数据查询 / 环境配置 / 权限管理

---

## 技术约束

- Vue 3 + `script setup`，JavaScript，Ant Design Vue 4.x
